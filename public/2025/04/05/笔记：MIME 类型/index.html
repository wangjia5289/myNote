笔记：MIME 类型

Last updated on 2025-06-28T18:58:56+08:00

MIME 类型用于表示文档、文件或数据的类型，告诉客户端如何解析响应的内容,常用于 HTTP 协议的 Content-Type 和 Accept 头，指明发送或接收的数据格式，一般格式为：type/subtype。以下是一些常见的 MIME 类型：

  1. ==文本类型（Text Types）==
    1. text/html：HTML 文档
    2. text/plain：纯文本
    3. text/css：CSS 样式表
    4. text/javascript 或 application/javascript：JavaScript 脚本
    5. text/csv：CSV 格式的文本文件
  2. ==应用程序类型（Application Types）==
    1. application/json：JSON 数据格式
    2. application/xml：XML 数据格式
    3. application/xhtml+xml：XHTML 文档
    4. application/pdf：PDF 文件
    5. application/zip：ZIP 压缩文件
    6. application/octet-stream：任意的二进制数据（常用于文件下载）
    7. application/msword：Microsoft Word 文档
    8. application/vnd.openxmlformats-officedocument.wordprocessingml.document：Word 文档（.docx）
    9. application/vnd.ms-excel：Microsoft Excel 文档
    10. application/vnd.openxmlformats-officedocument.spreadsheetml.sheet：Excel 文档（.xlsx）
  3. ==图像类型（Image Types）==
    1. image/jpeg：JPEG 图像
    2. image/png：PNG 图像
    3. image/gif：GIF 图像
    4. image/svg+xml：SVG 矢量图
    5. image/webp：WebP 图像格式
    6. image/bmp：BMP 图像
  4. ==音频类型（Audio Types）==
    1. audio/mpeg：MP3 音频
    2. audio/ogg：OGG 音频
    3. audio/wav：WAV 音频
    4. audio/webm：WebM 音频
  5. ==视频类型（Video Types）==
    1. video/mp4：MP4 视频
    2. video/ogg：OGG 视频
    3. video/webm：WebM 视频
    4. video/x-msvideo：AVI 视频
  6. ==字体类型（Font Types）==
    1. font/woff：WOFF 字体
    2. font/woff2：WOFF2 字体
    3. application/font-woff：另一种 WOFF 的 MIME 类型
    4. font/ttf：TrueType 字体（TTF）
    5. font/otf：OpenType 字体（OTF）
  7. ==其他常见类型==
    1. multipart/form-data：用于表单数据（文件上传）
    2. application/x-www-form-urlencoded：URL 编码表单数据
    3. application/javascript：JavaScript 代码
