### 分布式集群部署


开放 9092、9093 端口


4GB 内存和 64GB SSD 磁盘空间




### 安装  JDK 17

Kafka 4.0.0 要求 Java 17（或更高版本）
```
# 1. 检查当前 JDK 版本
java -version


# 2. 安装 JDK 177
sudo apt-get install openjdk-17-jdk
```
