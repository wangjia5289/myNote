# 补充


### 主线程、异步线程分别应该执行哪些操作

1. ==主线程应处理非耗时的小逻辑操作，确保应用响应迅速：==
	1. 字符串拼接
	2. 参数校验
	3. 数据封装
	4. 简单计算
2. ==异步线程应处理那些耗时的、阻塞的操作，这些操作通常涉及等待外部资源或大量计算：==
	1. 阻塞的网络请求（HttpClient）
		1. 如 HTTP 请求、API 调用等需要等待响应的操作
	2. 阻塞的数据库查询（JDBC）
	3. 文件读写
	4. CPU 密集型操作
		1. 复杂运算
		2. 大规模数据处理
			1. 压缩和解压、加密和解密、大型数据集处理
		3. 媒体转码
			1. 视频格式转换、音频格式转换
		4. 等等等等
